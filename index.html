Installation

To install the Sigmath framework, follow these steps:

            
                $ npm install sigmath

Quick Start

To quickly get started with the Sigmath framework, follow these steps:

  1. Install the framework (see Installation section for details)
  2. Create a new project directory
  3. Initialize a new Sigmath project in the directory using the sigmath init command
  4. Start the development server using the sigmath start command
